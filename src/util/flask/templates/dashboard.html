Welcome to the Dashboard

This dashboard provides an overview of the key features available in the system, including tools for configuring and running simulations, visualizing data, and submitting jobs to be processed.

Sections Covered in the Dashboard:

  • Runtime Prediction: This section offers runtime predictions for jobs once they reach 10% completion. The system calculates metrics such as the computed time, remaining time, and total runtime prediction, helping users monitor job progress.
  • Visualization Dashboard: This section includes a live Grafana dashboard that provides real-time data monitoring, including graphs, performance metrics, and other key indicators relevant to your simulations and jobs.
  • Experiment Environment: In this environment, users can set up simulation configurations by defining parameters for sources, servers, and sinks. This is where users can experiment with different setups before actually submitting jobs for execution. It's designed to let users configure all the necessary aspects of a simulation and test different scenarios.
  • Job Submission: After configuring the simulation in the experiment environment, users can submit jobs for processing. In the job submission section, users specify the number of compute nodes, replications, and select a pre-configured scenario. The job submission ensures that the simulation is executed on the available resources, with options for monitoring and managing job execution.

Explore the various sections for detailed features and functionality.
